NeoB2b - NeoDocs
================
Readmine: http://redmine.neoteric.eu:3010/projects/neob2b-neodocs


Quick-start
---
```bash
npm install
bower install
grunt build
grunt githooks // probably unnecessary - should invoke after npm install
grunt serve
```


Tests
---
- `grunt test` run unit tests
- `grunt coverage` run coverage
